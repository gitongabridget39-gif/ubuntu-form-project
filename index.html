Ubuntu Opinion Form

HTML forms are used to collect input from users in a web application. The information entered can then be processed and stored for later use.

Components of an HTML Form

Form
The main container that holds all form controls. Purpose: collects user input in one place. Attributes: action, method.
Label
Describes an input field for the user. Purpose: tells the user what to enter. Attribute: for.
Input
Accepts user data such as names, dates, and gender options. Purpose: captures typed or selected data. Attributes: type, id, name, value.
Textarea
Provides space for a longer response. Purpose: captures an opinion or comment. Attributes: id, name, rows, cols.
Fieldset
Groups related controls together. Purpose: organizes the form into sections.
Legend
Gives a title to each fieldset. Purpose: identifies a group of fields.
Button
Performs an action when clicked. Purpose: submit, clear, or cancel the form. Attribute: type.
