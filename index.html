Parts of an HTML Form

Form Part Purpose Main Attributes
<form> Defines the area that contains all form controls. action, method
<label> Gives a name to an input field. for
<input> Collects user data such as text, dates, and radio options. type, id, name, value
<textarea> Allows the user to enter longer text. id, name, rows, cols
<fieldset> Groups related form elements together. -
<legend> Provides a title
