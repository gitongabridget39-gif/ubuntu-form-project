HTML forms are used to collect input from users. The information entered can be processed by a web application and stored for later use.

Parts of an HTML Form

<form>
Defines the form area that contains all form controls. Common attributes: action, method.
<label>
Gives a name to an input field so the user knows what to enter. Common attribute: for.
<input>
Used to collect user data such as text, dates, and radio options. Common attributes: type, id, name, value.
<textarea>
Used when the user needs to enter longer text such as an opinion. Common attributes: id, name, rows, cols.
<fieldset>
Groups related form elements together.
<legend>
Gives a title to a fieldset.
<button>
Creates clickable buttons such as submit, clear, or cancel. Common attribute: type.
